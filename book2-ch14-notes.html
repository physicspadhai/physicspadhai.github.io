IntrinsicPure Si/Ge$n_e = n_h = n_i$-

p-typeTrivalent: B, AlHolesElectrons

$n_e n_h = n_i^2$ - Mass action law

2. p-n Junction Diode

Forward Bias: p to +ve, n to -ve. Depletion layer decreases. Current due to majority carriers.

Reverse Bias: p to -ve, n to +ve. Depletion layer increases. Small leakage current.

Barrier Potential:

Si = 0.7V, Ge = 0.3V

3. Rectifier

Half Wave: 1 diode. Efficiency = 40.6%. Ripple freq = Input freq

Full Wave: 2 or 4 diodes. Efficiency = 81.2%. Ripple freq = 2× Input freq

Filter: Capacitor used to smooth output

4. Transistor

3 terminals: Emitter, Base, Collector. Two types: n-p-n, p-n-p

$I_E = I_B + I_C$
$\alpha = I_C/I_E ≈ 0.98$
$\beta = I_C/I_B ≈ 50-300$

CE Amplifier: Voltage gain $A_v = \beta \frac{R_C}{R_B}$

Previous Year Questions 2026

CBSE 2024:

Q. Draw V-I characteristics of p-n junction in forward and reverse bias.

Ans: Forward: Exponential rise after knee voltage 0.7V Si. Reverse: Small μA current till breakdown.

CBSE 2023:

Q. For CE transistor, β = 100, $I_B = 20$ μA. Find $I_C$ and $I_E$.

Ans: $I_C = \beta I_B = 100×20$ μA = $2$ mA. $I_E = I_B + I_C = 2.02$ mA
