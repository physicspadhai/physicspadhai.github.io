1. Wavefront & Huygens Principle

Wavefront: Locus of points having same phase at given instant.

Types: 1. Spherical 2. Cylindrical 3. Plane

Huygens Principle:

  • Each point on wavefront acts as source of secondary wavelets
  • New wavefront = Tangent envelope to all secondary wavelets

2. Interference of Light - YDSE

Condition for Sustained Interference: Sources must be coherent, same frequency, constant phase diff.

Young's Double Slit Experiment:

Path difference: $\Delta x = \frac{yd}{D}$
Bright Fringe: $\Delta x = n\lambda$ → $y_n = \frac{n\lambda D}{d}$
Dark Fringe: $\Delta x = (2n-1)\frac{\lambda}{2}$ → $y_n = (2n-1)\frac{\lambda D}{2d}$

Fringe Width: $\beta = \frac{\lambda D}{d}$ → Same for all fringes

Angular Width: $\theta = \frac{\lambda}{d}$

3. Diffraction of Light

Bending of light around corners of obstacle. Size of obstacle ≈ $\lambda$

Single Slit Diffraction:

Central Maxima: $\theta = 0$
Minima: $a\sin\theta = n\lambda$, n = 1,2,3...
Secondary Maxima: $a\sin\theta = (2n+1)\frac{\lambda}{2}$

Width of Central Maxima: $2\theta = \frac{2\lambda}{a}$ → Widest & Brightest

Difference vs Interference: Diffraction = one slit, Interference = two slits

4. Polarisation

Restricting vibrations of light to one plane. Only transverse waves can be polarised.

Brewster's Law: When reflected & refracted rays ⊥, reflected light is completely plane polarised.

$\mu = \tan i_p$ → $i_p$ = Brewster angle/polarising angle

Malus Law: $I = I_0\cos^2\theta$ → $\theta$ = angle between polariser & analyser

Uses: Sunglasses, 3D movies, LCD screens, stress analysis

5. Resolving Power

Microscope: $R.P = \frac{2\mu\sin\theta}{\lambda}$

Telescope: $R.P = \frac{D}{1.22\lambda}$ → D = aperture diameter

Previous Year Questions 2026

CBSE 2024:

Q. In YDSE, separation between slits is 0.28 mm, screen 1.4 m away. Distance between central bright and 4th bright fringe is 1.2 cm. Find wavelength.

Ans: $y_4 = \frac{4\lambda D}{d}$ → $1.2×10^{-2} = \frac{4×\lambda×1.4}{0.28×10^{-3}}$ → $\lambda = 600$ nm

CBSE 2023:

Q. Why is diffraction of sound more evident than light in daily life?

Ans: Diffraction occurs when obstacle size ≈ $\lambda$. Sound $\lambda$ = 0.1m to 10m ≈ door size. Light $\lambda$ = 400-700nm, too small vs daily obstacles.

CBSE 2022:

Q. State Brewster's law. Find polarising angle for glass $\mu = 1.5$.

Ans: $\mu = \tan i_p$ → $1.5 = \tan i_p$ → $i_p = \tan^{-1}(1.5) = 56.3°$

CBSE 2020:

Q. Width of slit is made double. How will width of central diffraction band change?

Ans: Width $\propto \frac{1}{a}$. If a → 2a, then width becomes half.
