1. Huygens Principle

  • Each point on wavefront = source of secondary wavelets
  • New wavefront = tangent to all wavelets

2. Interference - YDSE

Condition for bright fringe: Path difference = $n\lambda$

Position of nth bright fringe: $y_n = \frac{n\lambda D}{d}$
Position of nth dark fringe: $y_n = \frac{(2n-1)\lambda D}{2d}$

Fringe Width:

$\beta = \frac{\lambda D}{d}$
D = distance to screen, d = slit separation

3. Diffraction at Single Slit

Central Maxima: $\theta = 0$

Minima: $a\sin\theta = n\lambda$, n = 1,2,3...

Width of central maxima = $\frac{2\lambda D}{a}$
a = slit width

4. Polarisation

Light waves are transverse. Polaroids transmit only one component.

Malus Law:

$I = I_0 \cos^2\theta$
θ = angle b/w pass axes

Brewster's Law:

$\tan i_p = \mu$
$i_p$ = Polarising angle

Previous Year Questions 2026

CBSE 2024:

Q. In YDSE, distance between slits is 0.28mm, screen is 1.4m away. Distance b/w central bright and 4th bright is 1.2cm. Find λ.

Ans: $y_4 = \frac{4\lambda D}{d}$ → $\lambda = \frac{y_4 d}{4D} = \frac{1.2×10^{-2} × 0.28×10^{-3}}{4×1.4} = 600$ nm

CBSE 2023:

Q. State Huygens principle. Use it to prove laws of reflection.

Ans: Each point on wavefront acts as secondary source. For reflection, incident & reflected wavefronts make same angle with surface → $i = r$
