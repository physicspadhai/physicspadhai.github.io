Chapter 7: Alternating Current 🔄⚡

1. AC Voltage and Current

Alternating current changes magnitude and direction periodically with time.

$$V = V_0 \sin\omega t$$ $$I = I_0 \sin\omega t$$ Where $V_0$ = Peak voltage, $I_0$ = Peak current
$\omega = 2\pi f$, f = 50 Hz in India

Mean/Average Value:

$$I_{avg} = \frac{2I_0}{\pi} = 0.637 I_0 \text{ for half cycle}$$ Full cycle ke liye $I_{avg} = 0$

RMS Value:

DC ka wo value jo AC ke equal heat produce kare.

$$I_{rms} = \frac{I_0}{\sqrt{2}} = 0.707 I_0$$ $$V_{rms} = \frac{V_0}{\sqrt{2}} = 0.707 V_0$$
🔥 Board Point: Ghar me 220V supply RMS value hoti hai. Peak value = $220\sqrt{2} = 311$ V

2. AC through Pure Resistor (R)

$$V = V_0 \sin\omega t$$ $$I = \frac{V_0}{R}\sin\omega t = I_0 \sin\omega t$$

Phase: V aur I same phase me. Phase difference = 0°

Phasor: V aur I same direction me

3. AC through Pure Inductor (L)

$$V = V_0 \sin\omega t$$ $$I = I_0 \sin(\omega t - \pi/2)$$ Where $I_0 = \frac{V_0}{X_L}$

Phase: Current lags voltage by 90° or $\pi/2$

Inductive Reactance:

$$X_L = \omega L = 2\pi f L$$ Unit: Ohm (Ω)

4. AC through Pure Capacitor (C)

$$V = V_0 \sin\omega t$$ $$I = I_0 \sin(\omega t + \pi/2)$$ Where $I_0 = \frac{V_0}{X_C}$

Phase: Current leads voltage by 90° or $\pi/2$

Capacitive Reactance:

$$X_C = \frac{1}{\omega C} = \frac{1}{2\pi f C}$$ Unit: Ohm (Ω)

PYQ 2023:

Q. Why capacitor blocks DC but allows AC?
Ans: For DC f=0, $X_C = 1/0 = \infty$. For AC, $X_C = 1/\omega C$ finite value.

5. LCR Series Circuit

$$V = V_0 \sin\omega t$$ $$I = I_0 \sin(\omega t + \phi)$$

Impedance (Z):

$$Z = \sqrt{R^2 + (X_L - X_C)^2}$$ $$I_0 = \frac{V_0}{Z}$$

Phase Angle:

$$\tan\phi = \frac{X_L - X_C}{R} = \frac{V_L - V_C}{V_R}$$
Condition Phase Circuit Nature
$X_L > X_C$ φ +ve, I lags V Inductive
$X_C > X_L$ φ -ve, I leads V Capacitive
$X_L = X_C$ φ = 0, I in phase Resistive

6. Resonance in LCR Circuit

When $X_L = X_C$, impedance minimum = R, current maximum.

$$\omega_0 L = \frac{1}{\omega_0 C}$$ $$f_0 = \frac{1}{2\pi\sqrt{LC}}$$ $f_0$ = Resonant frequency

At Resonance:

  • Z = R minimum
  • $I_0 = V_0/R$ maximum
  • Power factor = 1
  • Phase angle φ = 0

Quality Factor (Q):

$$Q = \frac{\omega_0 L}{R} = \frac{1}{\omega_0 CR} = \frac{1}{R}\sqrt{\frac{L}{C}}$$

PYQ 2024:

Q. L=2H, C=32μF, R=10Ω. Find resonant frequency.
Ans: $f_0 = \frac{1}{2\pi\sqrt{LC}} = \frac{1}{2\pi\sqrt{2 \times 32 \times 10^{-6}}} = 19.9$ Hz

7. Power in AC Circuit

$$P_{avg} = V_{rms} I_{rms} \cos\phi$$ $$\cos\phi = \frac{R}{Z} = \text{Power Factor}$$

Cases:

  • Pure R: φ = 0, $\cos\phi = 1$, P = $V_{rms}I_{rms}$ maximum
  • Pure L or C: φ = 90°, $\cos\phi = 0$, P = 0 Wattless current
  • LCR: P = $V_{rms}I_{rms}\cos\phi$

8. LC Oscillations

Energy oscillation between L and C. No loss if R = 0.

$$q = q_0 \cos\omega t$$ $$I = -I_0 \sin\omega t$$ $$\omega = \frac{1}{\sqrt{LC}}$$

9. Transformer

AC voltage ko step up ya step down karta hai. Principle: Mutual Induction

$$\frac{V_S}{V_P} = \frac{N_S}{N_P} = \frac{I_P}{I_S}$$ For ideal transformer: $V_P I_P = V_S I_S$

Efficiency:

$$\eta = \frac{P_{out}}{P_{in}} \times 100\% = \frac{V_S I_S}{V_P I_P} \times 100\%$$

Energy Losses:

  • Copper loss: $I^2R$ heating in windings
  • Iron loss: Eddy current + Hysteresis loss
  • Flux leakage: All flux not linked
